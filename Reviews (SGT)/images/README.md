# Reviewer photos (Google Business Profile)

Save each reviewer’s **profile photo** from Google here so paths in `Reviews (SGT).md` stay stable.

**Suggested naming**

- `reviewer-01.jpg` … `reviewer-05.jpg` — matches **`reviews-export-template.csv`** and **`astro-site/public/Media (SGT)/Reviews (SGT)/`** (copy the same files here so the repo stays in sync).

**How to get images (manual, from GBP)**

1. Open [Google Maps](https://maps.app.goo.gl/zPRAe4qAJ4sntCNh8) (or [Google Business Profile](https://business.google.com/) → Reviews).
2. Open each review; right‑click the reviewer avatar → **Save image as…** into this folder.
3. Update the **Image file** column in `Reviews (SGT).md` to match the filename you saved.

**Note:** Reviewer avatars are Google account images; keep use aligned with your marketing permissions and Google’s terms.
